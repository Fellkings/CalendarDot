# CalendarDot

**CalendarDot** — это удобное веб-приложение (веб-сервис) для планирования времени и ведения личного или корпоративного расписания. 

Приложение работает прямо в браузере как **SPA (Single Page Application)**. Благодаря использованию веб-сокетов, данные между клиентом и сервером обновляются мгновенно и без перезагрузки страниц.

---

## Основные возможности

* **Интуитивный интерфейс:** Современный дизайн в стиле Material Design 3 с поддержкой Светлой и Темной тем оформления.
* **Два режима отображения:**
  * **Режим месяца:** Удобная сетка для долгосрочного планирования с индикаторами задач.
  * **Режим недели:** Детализированная почасовая сетка времени (24 часа) для точной фиксации встреч и дел.
* **Категории задач:** Возможность группировать события по цветам и эмодзи-иконкам.
* **Быстрый поиск:** Полнотекстовый поиск по названиям и описаниям задач с автоматическим переходом на нужную дату.
* **Безопасность:** Пароли пользователей хранятся в базе данных в зашифрованном виде. Каждый пользователь видит только свое расписание.

---

## Стек технологий

* **Язык программирования:** Python
* **Интерфейс (Frontend):** Flet (на базе движка Flutter)
* **База данных:** SQLite (встраиваемая СУБД, хранит все данные в одном локальном файле)
* **Взаимодействие с БД (Backend):** SQLAlchemy ORM (обеспечивает безопасность транзакций и защиту от SQL-инъекций)

Локальный запуск проекта

Чтобы запустить веб-приложение на своем компьютере, выполните следующие простые шаги:
1. Клонирование репозитория

Скачайте проект к себе на компьютер:
Bash

git clone [https://github.com/Fellkings/CalendarDot.git](https://github.com/Fellkings/CalendarDot.git)
cd CalendarDot

2. Создание и активация виртуального окружения

    Для Windows:
    Bash

    python -m venv venv
    venv\Scripts\activate

    Для macOS / Linux:
    Bash

    python3 -m venv venv
    source venv/bin/activate

3. Установка библиотек

Установите необходимые для работы приложения компоненты:
Bash

pip install -r requirements.txt

4. Запуск приложения

Запустите главный управляющий скрипт:
Bash

python main.py

После этого приложение автоматически создаст файл базы данных calendardot.db и откроет интерактивное окно календаря в вашем веб-браузере по умолчанию.
